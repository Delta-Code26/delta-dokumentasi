    ‚ûó Operator di Java

    Operator digunakan untuk melakukan operasi pada variabel dan nilai. Java memiliki berbagai macam operator, termasuk operator aritmatika, perbandingan, logika, dan lainnya.

    üßÆ Operator Aritmatika

    Operator aritmatika digunakan untuk melakukan operasi matematika dasar.

    Operator Deskripsi Contoh
    + Penjumlahan 5 + 3
    - Pengurangan 5 - 3
    * Perkalian 5 * 3
    / Pembagian 6 / 3
    % Modulus (Sisa) 5 % 3

    Contoh:

    int a = 10;
int b = 5;

System.out.println(a + b); // Output: 15
System.out.println(a - b); // Output: 5
System.out.println(a * b); // Output: 50
System.out.println(a / b); // Output: 2
System.out.println(a % b); // Output: 0

    üí• Operator Penugasan (Assignment)

    Operator penugasan digunakan untuk memberikan nilai pada variabel.

    Operator Deskripsi Contoh
    = Penugasan dasar x = 5
    += Penambahan x += 3
    -= Pengurangan x -= 3
    *= Perkalian x *= 3
    /= Pembagian x /= 3

    Contoh:

    int x = 5;
x += 3; // x = x + 3
System.out.println(x); // Output: 8

    ‚öñÔ∏è Operator Perbandingan (Comparison)

    Operator perbandingan digunakan untuk membandingkan dua nilai.

    Operator Deskripsi Contoh
    == Sama dengan a == b
    != Tidak sama dengan a != b
    > Lebih besar a > b
    < Lebih kecil a < b
    >= Lebih besar atau sama dengan a >= b
    <= Lebih kecil atau sama dengan a <= b

    Contoh:

    int a = 10;
int b = 5;

System.out.println(a == b); // Output: false
System.out.println(a != b); // Output: true
System.out.println(a > b);  // Output: true

    üßë‚Äçü§ù‚Äçüßë Operator Logika (Logical)

    Operator logika digunakan untuk menggabungkan beberapa kondisi.

    Operator Deskripsi Contoh
    && AND (keduanya harus benar) a > b && b < 10
    || OR (salah satu benar) a > b || b > 10
    ! NOT (membalikkan kondisi) !(a > b)

    Contoh:

    boolean x = true;
boolean y = false;

System.out.println(x && y); // Output: false
System.out.println(x || y); // Output: true
System.out.println(!x);     // Output: false

    üõ†Ô∏è Operator Unary

    Operator unary digunakan untuk memanipulasi satu operand (nilai atau variabel).

    Operator Deskripsi Contoh
    ++ Penambahan 1 a++
    -- Pengurangan 1 a--
    + Positif (memastikan nilai positif) +a
    - Negatif (memastikan nilai negatif) -a

    Contoh:

    int a = 5;
a++; // a = a + 1
System.out.println(a); // Output: 6

a--; // a = a - 1
System.out.println(a); // Output: 5

    üîÑ Operator Ternary (Conditional)

    Operator ternary adalah cara singkat untuk menulis if-else.

    int a = 10;
String hasil = (a > 5) ? "Lebih besar dari 5" : "Kurang dari atau sama dengan 5";
System.out.println(hasil); // Output: Lebih besar dari 5

    üìÖ Operator Bitwise (Biner)

    Operator bitwise digunakan untuk operasi pada level biner (bit).

    Operator Deskripsi Contoh
    & AND a & b
    | OR a | b
    ^ XOR (exclusive OR) a ^ b
    ~ NOT (inversi) ~a

    Contoh:

    int a = 5;  // 0101 dalam biner
int b = 3;  // 0011 dalam biner
System.out.println(a & b); // Output: 1 (biner: 0001)

    üí° Tips Penggunaan

    • Gunakan operator perbandingan dengan hati-hati, pastikan tipe data yang dibandingkan sesuai.
    • Hindari penggunaan operator bitwise kecuali benar-benar diperlukan untuk optimisasi atau aplikasi tingkat rendah.
    • Gunakan operator ternary untuk kode yang lebih ringkas, namun hindari penggunaan yang berlebihan agar tidak membingungkan.

    üéì Latihan

    1. Buat program yang meminta dua angka dari user, lalu lakukan operasi penjumlahan, pengurangan, perkalian, dan pembagian.
    2. Coba implementasikan operator logika dengan kondisi yang lebih kompleks.
