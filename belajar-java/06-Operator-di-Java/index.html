    06. Operator di Java

    Operator adalah simbol atau karakter spesial yang digunakan untuk melakukan operasi pada variabel dan nilai. Java punya banyak jenis operator yang siap dipakai buat segala kondisi—dari hitung-hitungan sampai logika super ribet.

    ➕ Operator Aritmatika

    Operator Fungsi Contoh
    + Penjumlahan a + b
    - Pengurangan a - b
    * Perkalian a * b
    / Pembagian a / b
    % Modulus (sisa) a % b

    🧠 Catatan: Kalau kamu bagi int dengan int, hasilnya juga int (dibulatkan ke bawah).

    🧮 Contoh Aritmatika

    int a = 10, b = 3;
System.out.println(a + b);  // 13
System.out.println(a - b);  // 7
System.out.println(a * b);  // 30
System.out.println(a / b);  // 3
System.out.println(a % b);  // 1

    🔁 Operator Penugasan (Assignment)

    Operator Fungsi Contoh
    = Menetapkan nilai a = 5
    += Tambah lalu assign a += 2 ⇒ a = a + 2
    -= Kurang lalu assign a -= 2
    *= Kali lalu assign a *= 2
    /= Bagi lalu assign a /= 2
    %= Mod lalu assign a %= 2

    🔍 Operator Perbandingan (Relasional)

    Operator Fungsi Contoh
    == Sama dengan a == b
    != Tidak sama dengan a != b
    > Lebih besar a > b
    < Lebih kecil a < b
    >= Lebih besar atau sama a >= b
    <= Lebih kecil atau sama a <= b

    Hasil dari operator ini selalu boolean (true atau false).

    🔗 Operator Logika

    Operator Nama Contoh
    && AND a > 5 && b < 10
    || OR a > 5 || b < 10
    ! NOT !(a > 5)

    🧠 Contoh Penggunaan Logika

    int umur = 20;
boolean punyaSIM = true;

if (umur >= 17 && punyaSIM) {
    System.out.println("Boleh nyetir");
} else {
    System.out.println("Belum boleh nyetir");
}

    🧙 Operator Unary

    Operator Fungsi Contoh
    + Nilai positif +a
    - Nilai negatif -a
    ++ Increment (naik 1) a++ / ++a
    -- Decrement (turun 1) a-- / --a

    🔥 Operator Ternary (Cepat & Ringkas)

    String hasil = (nilai >= 75) ? "Lulus" : "Remedial";

    Artinya: kalau nilai >= 75, maka hasil = "Lulus", selain itu "Remedial".

    👀 Operator instanceof

    Cek apakah sebuah objek adalah instance dari class tertentu.

    if (objek instanceof String) {
    System.out.println("Ini adalah String");
}

    💡 Tips Cepat

    • Gunakan == untuk angka, equals() untuk membandingkan String.
    • Gunakan ++ dengan hati-hati — bisa beda antara ++a dan a++.
